# Summary

VoxCPM is an OpenBMB repository for tokenizer-free text-to-speech, with VoxCPM2 presented as the current flagship release. The repo positions VoxCPM2 as a 2B multilingual TTS model supporting 30 languages, voice design from natural-language prompts, controllable voice cloning, continuation-style high-fidelity cloning, and 48kHz output. It provides Python API, CLI, local web demo, documentation links, model links, and a production-serving pointer via Nano-VLLM.

The project is notable because it packages a broad feature set, claims commercial-ready Apache-2.0 licensing, and targets both research and practical deployment. For archiving purposes, the key value is its role as a public reference point in the recent open-source TTS landscape, especially around multilingual synthesis and controllable cloning.
