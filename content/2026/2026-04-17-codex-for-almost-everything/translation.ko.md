# 거의 모든 것을 위한 Codex

원문: https://openai.com/index/codex-for-almost-everything/
저장일: 2026-04-17
번역 메모: 직역보다 자연스러운 한국어 흐름을 우선해 다듬은 번역본.

OpenAI는 Codex에 대한 대규모 업데이트를 발표하며, 이를 소프트웨어 개발 라이프사이클 전반에서 매주 300만 명 이상의 개발자가 사용하는 더 강력한 파트너로 설명한다.

발표에 따르면 Codex는 이제 사용자의 컴퓨터를 함께 조작할 수 있고, 더 많은 앱과 도구와 연동되며, 이미지를 생성하고, 선호를 기억하고, 이전 행동에서 배우고, 반복적이거나 장기적인 작업도 맡을 수 있다. 또한 Codex 앱은 PR 리뷰, 여러 파일과 터미널 보기, SSH를 통한 원격 개발 박스 접속, 앱 내부 브라우저를 통한 프런트엔드 작업, 앱, 게임 반복 개선 같은 개발자 워크플로우 지원을 한층 더 강화했다고 한다.

## 컴퓨터 사용과 브라우저 사용

글에 따르면 Codex는 배경 컴퓨터 사용 기능을 통해 사용자의 컴퓨터 위에서 직접 보고, 클릭하고, 타이핑하며 앱을 다룰 수 있다. OpenAI는 여러 에이전트가 한 대의 Mac에서 동시에 작업하면서도 사용자의 다른 작업을 방해하지 않을 수 있다고 설명한다.

또한 Codex는 웹과도 더 자연스럽게 연동되기 시작했다고 한다. 앱 내부 브라우저에서 사용자가 페이지 위에 직접 코멘트를 달아 에이전트에 정밀한 지시를 줄 수 있으며, 현재는 프런트엔드 및 게임 개발에 특히 유용하고 앞으로는 브라우저 전체 제어로 확장할 계획이라고 설명한다.

## 이미지, 플러그인, 더 풍부한 워크플로우

글은 Codex가 이제 `gpt-image-1.5`를 활용해 이미지를 생성하고 수정할 수 있다고 설명한다. 스크린샷과 코드와 결합하면 제품 콘셉트, 프런트엔드 디자인, 목업, 게임 제작에 유용하다는 설명이다.

OpenAI는 또한 skills, 앱 연동, MCP 서버를 결합한 90개 이상의 추가 플러그인을 출시한다고 밝혔다. 예시로 Atlassian Rovo, CircleCI, CodeRabbit, GitLab Issues, Microsoft Suite, Neon by Databricks, Remotion, Render, Superpowers를 언급한다.

앱은 GitHub 리뷰 코멘트 대응, 여러 터미널 탭 실행, SSH를 통한 원격 개발 박스 연결을 알파 단계로 지원하며, PDF, 스프레드시트, 슬라이드, 문서 파일을 사이드바에서 풍부한 미리보기와 함께 열 수 있다고 한다. 또한 계획, 출처, 산출물을 추적하는 summary pane도 추가됐다고 설명한다.

## 자동화, 메모리, 선제적 제안

OpenAI는 자동화 기능을 확장해 기존 대화 스레드를 재사용할 수 있게 하면서 맥락을 더 오래 보존할 수 있다고 말한다. Codex는 미래 작업을 스스로 예약하고, 며칠 혹은 몇 주에 걸친 장기 과업을 이어서 수행하기 위해 자동으로 다시 깨어날 수도 있다고 설명한다.

글은 팀들이 이미 Slack, Gmail, Notion 같은 도구를 오가며 열려 있는 PR 처리, 후속 작업, 빠르게 변하는 대화 추적 등에 자동화를 활용하고 있다고 설명한다.

또한 memory 프리뷰도 공개했다. 이는 사용자 선호, 수정 이력, 시간 들여 모은 정보 같은 유용한 맥락을 Codex가 이전 경험에서 기억할 수 있게 하는 기능으로 설명된다. OpenAI는 이를 통해 다음 작업들이 더 빠르고 더 높은 품질로 완료될 수 있다고 말한다.

Codex는 이제 유용한 후속 작업을 선제적으로 제안할 수도 있다고 한다. 프로젝트 맥락, 연결된 플러그인, 메모리를 사용해 하루를 어디서 시작할지, 이전 프로젝트를 어디서 다시 이어갈지 추천할 수 있으며, Google Docs의 열린 코멘트, Slack과 Notion의 관련 정보, 코드베이스 맥락을 모아 우선순위가 정리된 작업 목록을 보여줄 수 있다는 설명이다.

## 배포와 포지셔닝

OpenAI는 이 업데이트가 지금부터 ChatGPT 계정으로 로그인한 Codex 데스크톱 앱 사용자에게 순차 배포된다고 밝혔다. 메모리와 맥락 인식 제안 같은 개인화 기능은 Enterprise, Edu, EU/UK 사용자에게도 곧 제공될 예정이며, 컴퓨터 사용 기능은 우선 macOS에서 시작하고 이후 EU/UK로 확대할 계획이라고 설명한다.

이 글은 이번 릴리스를, Codex가 단순 코드 작성 도구에서 시스템 이해, 맥락 수집, 작업 검토, 디버깅, 팀 협업, 장기 과업 지속까지 수행 범위를 넓혀가는 흐름의 일부로 제시한다.
