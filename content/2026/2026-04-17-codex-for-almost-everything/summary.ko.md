# 요약

OpenAI 발표의 핵심은 Codex가 단순 코딩 보조 도구에서 여러 도구와 워크플로우를 넘나드는 소프트웨어 에이전트 작업 공간으로 확장되고 있다는 점입니다. 이번 업데이트에는 컴퓨터 사용, 앱 내 브라우저, 이미지 생성, 추가 플러그인, 자동화, 메모리, 선제적 제안, 풍부한 파일 미리보기, PR 리뷰와 원격 개발 박스 접근 같은 개발자 기능 강화가 포함됩니다.

더 큰 흐름으로 보면 OpenAI는 Codex를 코드 생성 제품이 아니라, 맥락을 기억하고 작업을 다시 이어가며 장기 개발 과업을 계속 굴리는 지속형 에이전트 워크스페이스로 포지셔닝하고 있습니다.
