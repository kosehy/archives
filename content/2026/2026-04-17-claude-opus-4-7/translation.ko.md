# Claude Opus 4.7 소개

원문: https://www.anthropic.com/news/claude-opus-4-7
저장일: 2026-04-17
번역 메모: 직역보다 자연스러운 한국어 흐름을 우선해 다듬은 번역본.

Anthropic은 Claude Opus 4.7이 이제 일반 제공 상태가 되었으며, 특히 고급 소프트웨어 엔지니어링과 난도가 높은 장기 과제에서 Opus 4.6보다 눈에 띄는 향상이 있다고 설명한다.

발표에 따르면 사용자들은 이전에는 가까운 감독이 필요했던 더 어려운 코딩 작업까지 이제 Opus 4.7에 더 자신 있게 맡길 수 있다고 말한다. Anthropic은 이 모델이 복잡한 다단계 과업을 더 엄밀하고 일관되게 처리하고, 지시를 더 정확히 따르며, 결과를 보고하기 전에 스스로 출력 검증 방법을 더 잘 고안한다고 설명한다.

또한 발표문은 Opus 4.7이 시각 이해도 크게 좋아졌고 더 높은 해상도의 이미지를 다룰 수 있으며, 인터페이스, 슬라이드, 문서 같은 전문 작업에서 더 세련되고 창의적인 결과를 낸다고 말한다. Anthropic은 Opus 4.7이 여전히 Claude Mythos Preview보다 전반 능력은 좁지만, 여러 벤치마크에서는 Opus 4.6보다 더 나은 성과를 보인다고 설명한다.

## 사이버보안 포지셔닝과 안전장치

Anthropic은 이번 출시를 앞서 발표한 Project Glasswing과 연결해 설명한다. Mythos Preview는 당분간 더 제한적으로 유지할 것이며, Opus 4.7은 금지되거나 고위험인 사이버보안 활용 징후를 자동 탐지하고 차단하는 새로운 안전장치를 탑재하고 출시되는 첫 모델이라고 밝힌다.

Anthropic은 훈련 과정에서 Mythos Preview에 비해 Opus 4.7의 사이버 능력을 차등적으로 낮추려는 시도도 했다고 말한다. 또한 취약점 연구, 침투 테스트, 레드팀 같은 합법적 보안 업무에 쓰려는 전문가들을 Cyber Verification Program으로 초대한다고 밝힌다.

## 제공 범위와 가격

Anthropic은 Opus 4.7이 Claude 제품군, API, Amazon Bedrock, Google Cloud Vertex AI, Microsoft Foundry 전반에서 이용 가능하다고 말한다. 가격은 Opus 4.6과 동일하게 입력 토큰 백만 개당 5달러, 출력 토큰 백만 개당 25달러라고 설명한다.

## 초기 테스터들의 주장

발표문은 초기 접근 테스터들의 인용으로 강하게 뒷받침된다. 이 추천사들에서 반복적으로 나타나는 주장은 다음과 같다.

- 어려운 다단계 코딩 작업에서 더 강한 성능
- 더 강한 장기 자율성
- 더 나은 지시 이행
- 더 적은 툴 오류
- 누락된 데이터나 한계에 대해 더 솔직한 태도
- 고해상도 시각 자료에 대한 더 나은 멀티모달 이해
- 비동기 워크플로, 에이전트, 코드 리뷰, 엔터프라이즈 문서 분석에서 더 강한 성능

글은 금융기술, 리서치 에이전트, 개발자 도구, 코드 리뷰, 대시보드 제작, 법률 툴링, 엔터프라이즈 문서 분석, 자율 침투 테스트 등 다양한 회사와 제품의 평가를 나열한다.

## Anthropic 자체 테스트에서 강조한 점

Anthropic은 사전 출시 테스트에서 몇 가지 내부 관찰 결과를 강조한다.

- Opus 4.7은 지시를 훨씬 더 잘 따르며, 그 결과 이전 모델에 맞춰 작성된 프롬프트는 새 모델에서 다시 조정이 필요할 수 있다. 예전 모델은 지시를 느슨하게 해석하거나 일부를 건너뛰었지만, Opus 4.7은 더 문자 그대로 따르기 때문이라는 설명이다.
- 더 높은 해상도의 이미지 입력을 지원하며, 긴 변 기준 2,576픽셀까지 처리할 수 있다고 말한다. 이는 이전 Claude 모델 대비 3배 이상 많은 시각 정보를 다룬다는 설명이다.
- 실제 금융 분석, 더 전문적인 프레젠테이션, 데이터 기반 에이전트 추론에서 성능이 더 좋다고 주장한다.
- 파일시스템 기반 메모리도 개선되어, 여러 세션에 걸친 장기 작업에서 중요한 메모를 더 잘 유지한다고 설명한다.

## 안전성과 정렬

Anthropic은 전반적 안전성 프로필은 Opus 4.6과 비슷하며, deception, sycophancy, misuse 협조 같은 우려 행동 비율이 낮다고 설명한다. 일부 항목, 예를 들어 정직성이나 prompt injection 저항성은 더 좋아졌다고 말하지만, 통제 물질에 대한 과도한 위해 감소 조언처럼 일부 항목에서는 약간 약해졌다고도 설명한다.

정렬 평가는 이 모델을 대체로 잘 정렬되고 신뢰할 만하지만, 행동 면에서 완전히 이상적이지는 않다고 설명한다. 또한 Anthropic은 자사 평가 기준으로 Mythos Preview가 여전히 가장 잘 정렬된 모델이라고 밝힌다.
