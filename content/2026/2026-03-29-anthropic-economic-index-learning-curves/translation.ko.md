# Anthropic Economic Index report: Learning curves

- 원문: https://www.anthropic.com/research/economic-index-march-2026-report
- 저장일: 2026-03-29
- 번역 메모: 직역보다 자연스러운 한국어 흐름을 우선해 다듬은 번역본.

Anthropic Economic Index는 Claude가 경제 전반에서 어떻게 사용되고 있는지를 프라이버시를 보존하는 분석 방식으로 추적하는 프로젝트다. 목적은 AI의 경제적 영향을 가능한 한 이른 시점에 파악해 연구자와 정책 담당자가 대비할 시간을 확보하는 데 있다.

이번 보고서는 2026년 2월의 Claude 사용 데이터를 다룬다. 표본 기간은 2월 5일부터 12일까지이며, Claude Opus 4.5 출시 후 3개월 시점이자 Claude Opus 4.6 출시와 겹치는 시기다.

보고서는 먼저 이전 리포트 이후 사용 양상이 어떻게 달라졌는지 정리한다. Claude.ai와 API 모두에서 보강적 사용, 즉 사용자의 역량을 보완하는 협업형 사용이 소폭 늘었다. Claude.ai에서는 상위 10개 작업이 차지하는 비중이 줄어들며 사용 사례가 더 다양해졌다. 그 결과 Claude.ai에서 다뤄지는 평균 작업의 임금 수준은 이전보다 약간 낮아졌다.

이후 보고서는 Claude가 노동시장과 경제 전반에 미치는 영향을 가르는 중요한 요소로 ‘학습 곡선’에 주목한다. 오래 사용한 이용자들은 Claude의 능력을 더 잘 끌어내는 습관과 전략을 익힌 것으로 보이며, 더 가치 있는 작업에 도전할 뿐 아니라 실제 대화에서 성공적인 응답을 끌어낼 확률도 더 높다는 점을 보여준다.

## 지난 리포트 이후 달라진 점

- Claude.ai의 사용 사례가 더 다양해졌다. 코딩 작업은 계속해서 Claude.ai의 보강적 사용에서 API 기반의 더 자동화된 워크플로우로 이동하고 있다. Claude.ai에서 상위 10개 작업이 차지하는 비중은 2025년 11월 24%에서 2026년 2월 19%로 줄었다.
- Claude의 채택은 더 낮은 임금 수준의 작업으로 확장됐다. 스포츠, 상품 비교, 집수리 같은 개인적 질문이 늘면서 Claude.ai에서 수행되는 평균 작업 가치가 약간 낮아졌다. 이는 초기 채택자는 코딩 같은 고부가가치 사용에 집중하고, 후발 채택자는 더 넓은 범위의 작업을 가져오는 전형적 채택 곡선과 맞물린다.
- 글로벌 사용의 불평등은 계속됐다. 상위 20개 국가가 전체 1인당 사용량의 48%를 차지해, 이전의 45%보다 더 높아졌다. 반면 미국 내에서는 상위 10개 주가 차지하는 비중이 40%에서 38%로 줄며 주별 격차는 다소 좁혀졌다.

## 학습 곡선

Anthropic Economic Index의 핵심 발견 중 하나는 초기 Claude 채택이 매우 불균등하다는 점이다. Claude는 고소득 국가, 미국 내에서는 지식노동자가 많은 지역, 그리고 제한된 전문 직무와 작업에 훨씬 더 집중적으로 사용된다.

여기서 중요한 질문은 이런 채택 격차가 AI의 혜택이 누구에게, 어디에 돌아갈지를 어떻게 결정하느냐는 것이다. 만약 효과적인 AI 사용에 보완적 기술과 전문성이 필요하고, 그런 기술이 실제 사용과 실험을 통해 습득될 수 있다면, 초기 채택의 이점은 자기강화적으로 확대될 수 있다.

보고서는 사용자가 모델 능력을 작업에 어떻게 맞추는지, 그리고 플랫폼 경험이 쌓이면서 사용 패턴과 성과가 어떻게 달라지는지를 살펴본다.

- 모델 선택은 작업에 맞춰진다. 사용자는 더 높은 임금을 받는 작업에 Anthropic의 가장 고성능 모델군인 Opus를 더 자주 선택한다. 유료 Claude.ai 사용자 기준으로 코딩 작업에서는 Opus 사용 비율이 평균보다 높고, 튜터링 관련 작업에서는 평균보다 낮다. 이 차이는 API 사용자에게서 더 뚜렷하다.
- 사용 기간이 길수록 성공률도 높다. 6개월 이상 Claude를 써온 사용자는 개인적 사용은 줄고, 더 높은 교육 수준과 관련된 작업을 더 많이 수행한다. 또한 작업 종류, 국가, 기타 요인을 통제해도 대화 성공률이 더 높다. 이는 초기 채택자의 성향 때문일 수도 있지만, 실제 사용을 통한 학습 효과를 반영할 가능성도 있다.

## Claude.ai 사용 사례의 다양화

Anthropic은 Claude.ai와 자사 API에서 각각 100만 개의 대화를 표본으로 분석했다. 코딩은 여전히 가장 흔한 사용 사례이며, Claude.ai 대화의 35%를 차지한다. 하지만 2025년 11월부터 2026년 2월 사이, Claude.ai에서 상위 10개 작업의 비중은 24%에서 19%로 낮아졌다.

이 변화의 일부는 Claude Code 같은 도구를 통해 코딩 작업이 API 워크플로우로 이동했기 때문이다. Claude Code는 에이전트형 구조 때문에 코딩 과업을 더 잘게 나눈 API 호출들로 분해하고, 이는 개별 작업으로 분류된다. 그러나 진짜 변화는 사용자 구성의 변화이기도 하다. coursework 비중은 19%에서 12%로 줄었고, 개인적 사용은 35%에서 42%로 늘었다. 학사 일정과 캐주얼 사용자 증가가 주요 배경으로 제시된다.

작업이 다양해졌다고 해도 완전히 새로운 작업이 대거 등장한 것은 아니다. 이전 리포트에서 추정했던 “약 49%의 직업에서 업무의 최소 4분의 1이 Claude로 수행된 적 있다”는 수치는 거의 변하지 않았다.

## 협업과 자동화 패턴

Anthropic은 상호작용을 크게 자동화와 보강으로 나눈다. Claude.ai에서는 validation과 learning 패턴이 늘며 보강적 사용이 약간 증가했다. API 데이터에서는 자동화 비중이 줄었지만, 여전히 전반적으로는 인간이 덜 개입하는 지시형 워크플로우가 더 많다.

API 플랫폼은 Computer and Mathematical 범주의 작업 비중을 계속 늘리고 있다. 2025년 8월 이후 해당 범주 작업은 API에서 14% 증가했고 Claude.ai에서는 18% 감소했다. Anthropic은 이런 이동이 관련 직무의 더 임박한 업무 변화를 시사할 수 있다고 본다.

특히 최근 API에서 두드러지게 늘어난 워크플로우는 다음과 같다.
- 영업 및 아웃리치 자동화
- 자동화된 트레이딩 및 시장 운영

## 지리적 수렴과 확산 격차

미국 내에서는 주별 1인당 Claude 사용량 격차가 계속 줄어들고 있지만, 이전보다 속도는 느려졌다. 2025년 8월부터 2026년 2월 사이 상위 5개 주가 차지하는 비중은 30%에서 24%로 떨어졌다.

반면 국가 간으로 보면 사용은 오히려 조금 더 집중됐다. 상위 20개 국가는 전체 인구 조정 사용량의 48%를 차지해, 저채택 국가들이 상대적으로 더 뒤처지고 있음을 보여준다.

## AI를 배우는 방식

보고서는 AI 학습의 두 신호에 주목한다. 하나는 모델 선택이고, 다른 하나는 사용자 사용 기간이다.

Anthropic은 사용자가 더 어렵고 더 가치 있는 작업에 Opus를 배정하는 경향을 확인했다. 유료 Claude.ai 계정에서는 코딩 관련 작업의 Opus 사용 비중이 교육 관련 작업보다 높다. Claude.ai와 API 모두에서 더 높은 임금을 받는 직무와 연관된 작업일수록 Opus 비중이 높고, 이 효과는 API에서 훨씬 더 강하다.

사용 기간 측면에서, 6개월 이상 Claude를 써온 사용자는 Claude를 더 협업적으로 사용하고, 업무용으로 더 많이 활용하며, 개인적 사용은 줄이고, 더 복잡한 작업을 가져오며, 성공률도 더 높다.

물론 보고서는 이 차이의 일부가 코호트 효과나 생존자 편향일 수 있다고 인정한다. 초기 채택자가 본래 더 기술적일 수 있고, 장기 사용자는 원래 Claude에서 가치를 잘 뽑는 사람들일 수 있기 때문이다. 그럼에도 작업 유형, 모델, 사용 목적, 국가 등을 통제한 뒤에도 장기 사용자 집단은 더 높은 성공률을 보인다.

## 논의

Anthropic의 결론은 Claude가 여전히 미국 경제 평균을 대표하는 작업보다는 상대적으로 고부가가치이고 복잡한 작업에 더 많이 쓰인다는 것이다. 다만 사용자 기반이 넓어지면서, 보수가 낮고 더 캐주얼한 작업 비중도 조금씩 커지고 있다.

경험이 많은 사용자일수록 Claude를 더 협업적으로 사용하고, 더 업무 중심적인 이유로 사용하며, 더 복잡한 작업을 가져오고, 더 높은 성공률을 보인다. Anthropic은 이를 learning-by-doing, 즉 실제 사용을 통해 AI 활용 능력이 향상되는 현상과 일치한다고 본다.

이런 차이는 노동시장 불평등을 더 키울 수 있다. 기술적이고 고숙련 과업을 가진 초기 채택자가 AI에 더 먼저 노출되고, 동시에 더 잘 활용해 이득도 더 크게 얻는다면, 초기 채택의 이점은 시간이 갈수록 더 강화될 수 있기 때문이다.
