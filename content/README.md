# Public Archive

This folder contains article files approved for public publishing.

## Publishing rule

Only include files that are safe for public release.

Typical public files:
- `article_index.md`
- `original.<lang>.md`
- `translation.<lang>.md`
- `summary.en.md`
- `summary.ko.md`

Do not publish by default:
- `opinion.*.md`
- `personal-note.*.md`
- other private working notes unless explicitly approved
